# Achieving residential energy targets in the EU

Code and data for the paper:

> **Achieving residential energy targets in the EU: A multi country comparison
> of regulatory and economic instruments.**
> Ece Özer, Francesca Conselvan, Daniel Harringer, Andreas Müller, Lukas Kranzl.
> *Energy Policy* (under review), 2026.

The study compares four residential building policy packages under two primary
energy factor (PEF) trajectory assumptions across seven heating dominated EU
Member States (Finland, France, Germany, Ireland, Poland, Romania, Sweden),
using the Invert/Opt building stock model. It quantifies how the choice of
policy instruments and PEF assumptions jointly determine compliance with the
EU Energy Performance of Buildings Directive (EPBD, 2024/1275) residential
targets at 2030 and 2035.

## Reproducing the figures

Install the dependencies (Python 3.9+):

```bash
pip install -r requirements.txt
```

Then run the scripts (each writes its output into `figures/`):

```bash
cd scripts
python create_figures.py            # FED carrier mix, sPED trajectories, driver decomposition (2030 & 2035)
python create_fed_baseline_figure.py # 2020 baseline FED mix by carrier
python plot_target_heatmap.py        # 2030 and 2035 target-achievement heatmaps
```

`create_figures.py` reads `data/output_1_all.xlsx` and
`data/output_1_existing.xlsx` (falling back to the `target_indicators_2_*`
files for any missing country/scenario), and reproduces:

- `fig_total_fed_carrier` — residential final energy demand by carrier
- `fig_specific_ped_residential` — sPED reduction trajectories (2020–2050)
- `fig_waterfall_scen_2030` / `fig_waterfall_scen_2035` — driver decomposition
  (the script also writes a 2050 panel, which is not used in the paper)

## Figure-to-source map

| Figure | Generated by | Data |
|--------|--------------|------|
| `fig_total_fed_carrier` | `scripts/create_figures.py` | `output_1_all.xlsx` |
| `fig_specific_ped_residential` | `scripts/create_figures.py` | `output_1_*` |
| `fig_waterfall_scen_2030/2035` | `scripts/create_figures.py` | `output_1_*` (+ `target_indicators_2_*` fallback) |
| `fig_fed_baseline_2020` | `scripts/create_fed_baseline_figure.py` | `output_1_all.xlsx` |
| `fig_target_heatmap_2030/2035` | `scripts/plot_target_heatmap.py` | `output_1_all.xlsx` |
| `fig_country_map_annotated` | `scripts/create_country_map.py` | see note below |
| `fig_epbd_timeline` | hand-drawn (no script) | — |
| `fig_invert_model` | hand-drawn (no script) | — |

The MEPS thresholds table is derived from `data/dist_curve.xlsx`; the PEF values
table from `data/PEFs_input_EPBDwise_EU27_2025-12-09.csv`.

### Note on the country map

`scripts/create_country_map.py` requires `geopandas` and the Natural Earth
**`ne_10m_admin_0_map_subunits`** shapefile, which is not redistributed here.
Download it from https://www.naturalearthdata.com/ and place the unzipped
shapefile at `/tmp/ne_subunits/ne_10m_admin_0_map_subunits.shp` (or edit the
path near the top of the script). The final map (`figures/fig_country_map_annotated.pdf`)
is included so the repository is self-contained without re-running this step.
